// 01 — The Problem

Why is Time Series Data Uniquely Hard?

📉

Non-Stationarity

Statistical properties shift over time — mean, variance, and distribution all drift. Models trained on past windows fail silently on future windows.

🔗

Long-Range Dependencies

An event from 2 years ago may strongly influence today. Traditional windowed models can't reach far enough to capture these causal chains.

⏱️

Irregular Sampling

Sensor data, financial ticks, medical readings — real-world time series arrive at uneven intervals, breaking fixed-stride assumptions.

// 03 — Transformer Intuition

Self-Attention: Every Token Talks to Every Token

In a transformer, every position can directly attend to every other position in a single operation — regardless of distance.

Attention(Q,K,V) = softmax(QKᵀ/√dₖ)·V

Unlike RNNs that process tokens sequentially, attention computes all pairwise relationships in parallel — enabling both faster training and longer context windows.

// 06 — Attention Deep Dive

The Q / K / V Mechanism — Step by Step

HOVER CELLS — attention score heatmap

① Input Projection

Each patch token X is linearly projected into three matrices: Query (Q), Key (K), and Value (V) using learned weight matrices W_Q, W_K, W_V.

② Score Computation

Attention scores = QKᵀ. Each query vector "asks" how relevant every key vector is. High dot product = high relevance between two time positions.

③ Scaled Softmax

Scores are divided by √dₖ (prevents gradient saturation) then passed through softmax, yielding a probability distribution over all time positions.

④ Weighted Aggregation

The final output is the softmax weights multiplied by V (values). Each output position is a weighted average of all input positions' information.

// 08 — Feed-Forward & LayerNorm

Stabilizing Deep Temporal Networks

Feed-Forward Network (FFN)

After attention, each token passes independently through a 2-layer MLP: FFN(x) = max(0, xW₁ + b₁)W₂ + b₂. The intermediate dimension is typically 4× the model dimension (e.g., 512→2048→512). This adds non-linearity that attention alone cannot provide.

Layer Normalization

LayerNorm normalizes across feature dimensions (not batch): LN(x) = γ·(x−μ)/σ + β. Applied before or after each sub-layer (pre-norm preferred in large models). Prevents internal covariate shift and enables training of 100-layer networks.

Residual Connections

Output(x) = x + Sublayer(LN(x)). The residual shortcut ensures gradients flow cleanly to early layers, making 24–48 layer transformers trainable and stable.

// 11 — Pretraining & Fine-Tuning

From Masked Patches to Zero-Shot Forecasting

🎭 Masked Patch Modeling

Like BERT for text, randomly mask 40–75% of patch tokens during pretraining. The model learns to reconstruct masked patches from visible context — forcing it to learn temporal dependencies.

▲ Animated: masked (grey) patches being predicted from visible (teal) context

⚡ Zero-Shot vs Fine-Tuned

1

Pretrain: Train on massive diverse corpus with reconstruction objective. Model learns universal temporal representations.

2

Zero-Shot: Apply directly to new dataset without any retraining. Works if target domain overlaps with pretraining data distribution.

3

Fine-Tune: Continue training on domain-specific data with small learning rate. Adapts representations to target distribution.

4

Prompt-Tune: Freeze backbone, train only a small prefix of learnable tokens. Efficient for downstream tasks with limited data.

// 13 — Failure Modes

When Large TST Models Struggle

🌊

Distribution Shift

If the target data distribution differs significantly from pretraining (different seasonality, scale, or regime), zero-shot performance degrades sharply.

💡 Mitigation: Domain-adaptive fine-tuning or normalization layers
📏

Context Length Limits

Even with 512K context, many industrial datasets span years at 1-second resolution (billions of points). Quadratic attention complexity makes full-context inference prohibitive.

💡 Mitigation: Hierarchical patching, sparse attention, linear attention
💰

Computational Cost

A 200B parameter model requires 400GB+ of GPU memory for inference. Real-time edge deployment is currently impractical for most industrial applications.

💡 Mitigation: Quantization, distillation to smaller models

// 17 — Summary

Key Takeaways & Resources

1

Patch tokens reduce sequence length and enable efficient attention over long temporal contexts.

2

Self-attention captures arbitrary long-range dependencies in O(1) sequential operations vs O(n) for RNNs.

3

Multi-head attention allows simultaneous capture of daily, weekly, and seasonal patterns in separate subspaces.

4

Masked pretraining on trillion-token corpora enables zero-shot generalization across domains and datasets.

5

Scale matters — but so do distribution shift, context limits, and compute cost for real deployment.

6

Not always the answer — for small, stationary, well-characterized datasets, simpler models often win.
